Hi folks, let's continue to build our OFMW Development Environment.

Now we are going to install JAVA. First of all download the latest JDK on the Oracle website. In this case is the jdk-7u72-linux-x64.rpm

Run the command

sudo rpm -ivh  jdk-7u72-linux-x64.rpm

After that JAVA will be installed on your machine. Update your PATH file by opening

 /etc/profile

file by this command:

vim /etc/profile

and paste the following at the end of the file:

JAVA_HOME=/usr/local/java/jdk1.7.0_72 
PATH=$PATH:$HOME/bin:$JAVA_HOME/bin 
export JAVA_HOME 
export PATH

Now you have to tell the system that the new Oracle Java version is available by the following commands:

sudo update-alternatives --install "/usr/bin/java" "java" "/usr/java/jdk1.7.0_72/bin/java" 1
sudo update-alternatives --install "/usr/bin/javac" "javac" "/usr/java/jdk1.7.0_72/bin/javac" 1
sudo update-alternatives --install "/usr/bin/javaws" "javaws" "/usr/java/jdk1.7.0_72/bin/javaws" 1

 

Then  make sure you make Oracle Java JDK as default by this following commands:

sudo update-alternatives --set java /usr/local/java/jdk1.7.0_72/bin/java 
sudo update-alternatives --set javac /usr/local/java/jdk1.7.0_72/bin/javac 
sudo update-alternatives --set javaws /usr/local/java/jdk1.7.0_72/bin/javaws

Reload your system PATH /etc/profile by running this command:

source /etc/profile

Reboot your system and after that check your version of JAVA by this command:

java -version
 java version "1.7.0_72" 
Java(TM) SE Runtime Environment (build 1.7.0_72-b26) 
Java HotSpot(TM) Server VM (build 25.20-b23, mixed mode)

That's all.
